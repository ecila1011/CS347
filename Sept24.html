I wanted to find something cool using css that I could potentially use in my project. I've looked at animations before for my blog posts, but none seemed like they would be something useful enough for my website. So I went looking for something a bit different and found this article It showed some sample code for how to make an animation for some words that seemingly appeared from nowhere. I thought that was a cool technique so I copied the code and then tried to expand it a bit and played with some other ways for the words to appear. It wasn't anything too crazy technical, but it gave me a good jumping off point for some animations that I could utilize for my project.

Here is the code in my css file that I used to create the animations above. Again, nothing too crazy difficult but I like that it is relatively straightforward so that it will be easy to implement if I choose to use it for my project. 

/* oh */
.parent1 {
    height:5em;
    overflow-y: hidden;
    display: block;
    }

.parent1 .child1 {
    animation-name: appear;
    animation-duration: 5s;
    font-size: 5em;
    }

@keyframes appear {
    from { transform: translateY(1.8em) }
    to { transform: translateY(0) }
    }

/* hello */
.parent2 {
    height:5em;
    overflow-y: hidden;
    display: block;
    }
    
.parent2 .child2 {
    animation-name: slide;
    animation-duration: 5s;
    font-size: 5em;
    }
    
@keyframes slide { 
      from { transform: translateX(-1.8em) }
      to { transform: translateX(0) }
    }
